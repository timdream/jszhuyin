    關於 JS 注音

    JS 注音

    JSZhuyin 是完全由 JavaScript 寫成的自動選字注音輸入法。由 Mozilla 贊助開發，內建於 Gaia (Boot to Gecko 手機作業系統的 front-end)。

    作者

    timdream

    系統需求

    Gaia 需要使用 Firefox Nightly 或是 B2G runtime 執行。JSZhuyin 與此示範用介面則目標支援所有現代瀏覽器。

    在提供新實作的 IndexedDB 的平台會將詞庫存進 IndexedDB，加快載入速度（截至 2012 年 3 月只有 Firefox 有此功能）。

    程式碼

    詞庫

    使用同為 MIT 授權的小麥注音詞庫，由 mjhsieh 維護，基於 libtabe。

    使用問題

    基本使用方法

    打字，方向鍵的下可以開始選字，選字模式下可以用數字選字與左右方向鍵換頁。選到的項目不是整個緩衝區的話，可以繼續選下去選到完為止（或是按方向鍵上鍵離開選字模式）。

    為何互動不像正常的桌機智慧注音？

    因為 JS 注音是手機輸入法 …… 想知道他在手機上互動起來的樣子，請用 Firefox 去 Gaia Keyboard Demo 看看。

    有沒有其他相似的工具？

    線上輸入法很多，但實作方式都是把輸入送到伺服器去處理。這是用 Javascript 在瀏覽器實作自動選字輸入法，這是第一個（有錯的話還請告知）。

    斷詞的方法是？

    窮舉法與積分比較（笑）。窮舉是用很簡單的 Composition of a natural number，積分是小麥注音詞庫提供的。不過因為是窮舉，如果有正確的詞庫應該可以做其他的 phonetic IME。
